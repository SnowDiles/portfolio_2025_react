# 🌐 Portfolio - Next.js

## 📝 Présentation

Ce projet est un portfolio développé avec **Next.js**, permettant de présenter un profil professionnel sous la forme d'un site web interactif. Il est conçu pour offrir une expérience fluide et optimisée aux utilisateurs, avec une mise en page moderne et responsive. Grâce à ses fonctionnalités avancées, il facilite la mise en valeur de compétences, d'expériences et de réalisations tout en offrant un moyen simple et efficace de communication via un formulaire de contact.

Ce portfolio tire parti des avantages de **Next.js**, notamment le rendu côté serveur (SSR) et la génération de pages statiques (SSG), ce qui améliore à la fois les performances et l'optimisation pour les moteurs de recherche (SEO). Son code est modulaire et bien structuré, garantissant ainsi une maintenance simplifiée et une évolutivité optimale.

## ✨ Fonctionnalités

- 🎨 **Interface moderne et responsive** : compatible avec tous les types d'écrans (ordinateurs, tablettes, mobiles).
- 📩 **Envoi d'e-mails** via un formulaire de contact sécurisé, permettant aux visiteurs d'envoyer des messages facilement.
- 🔍 **Optimisation SEO** intégrée pour améliorer la visibilité du site sur les moteurs de recherche.
- ⚡ **Chargement rapide** grâce au rendu côté serveur (SSR) et à la génération de pages statiques (SSG).
- 🏗️ **Code modulaire et maintenable**, facilitant l'ajout de nouvelles fonctionnalités et la mise à jour du projet.
- 🔐 **Sécurisation des requêtes** pour éviter les spams et les abus via le formulaire de contact.
- 🌍 **Multilingue prêt** : le projet peut être facilement adapté pour supporter plusieurs langues.

## 📥 Installation

1. **Cloner le dépôt** :
   ```bash
   git clone https://github.com/SnowDiles/portfolio_perso_2025.git
   cd portfolio_perso_2025
   ```
2. **Installer les dépendances** :
   ```bash
   npm install
   ```
3. **Configurer les variables d'environnement** :
   - Créer un fichier `.env.local` à la racine du projet.
   - Ajouter les informations nécessaires pour la configuration des emails.

## 🚀 Lancer le projet

1. **Démarrer le serveur de développement** :
   ```bash
   npm run dev
   ```
2. **Accéder au portfolio** en ouvrant l'URL suivante dans un navigateur :
   ```
   http://localhost:3000/portfolio
   ```

### 🔄 Déploiement

Le projet peut être déployé facilement sur des plateformes comme **Vercel** ou **Netlify**. Voici les étapes pour un déploiement sur Vercel :
1. Installer l'outil CLI de Vercel si ce n'est pas déjà fait :
   ```bash
   npm install -g vercel
   ```
2. Lancer la commande de déploiement :
   ```bash
   vercel
   ```
3. Suivre les instructions pour finaliser l'installation.

## 🛠️ Technologies utilisées

- ⚛️ **Next.js** - Framework React pour le rendu côté serveur et le SSG.
- 🚀 **React** - Bibliothèque JavaScript pour la construction d'interfaces utilisateur dynamiques.
- ✉️ **Nodemailer** - Service permettant l'envoi d'e-mails depuis le site.
- 🎨 **TAILWIND** - Pour la personnalisation du design et l'amélioration de l'expérience utilisateur.
- 🔧 **Vercel** - Solution recommandée pour l'hébergement et le déploiement rapide du projet.
